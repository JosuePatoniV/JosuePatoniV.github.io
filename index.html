Sistema de administración de licencias de software

Acerca de: El sistema está pensado para administrar la compra, renovación y cancelación de licencias de software.
El administrador realiza algunas de estas acciones que quedan registradas en una base de datos y esto mismo ayuda a avisar oportunamente
a pagar a tiempo las licencias necesarias o cancelar otras. Esto permite una mejor gestión. Las licencias están vinculadas a uno o varios
equipos de cómputo. Al tener un registro facilita aclaraciones y hace más fácil la tarea de administración de licencias de software.
